# Google Play Store Assets

This directory contains RadarLite's Play Store submission copy, policy notes, and generated image assets.

## Public URLs

- App page: `https://luckyshot.github.io/radarlite/`
- Privacy policy: `https://luckyshot.github.io/radarlite/privacy.html`
- Source code: `https://github.com/luckyshot/radarlite`

## Files

- `google-play-listing.md`: store listing copy, categories, declarations, and review notes.
- `images/app_icon_512.png`: Play Console app icon.
- `images/feature_graphic_1024x500.png`: Play Console feature graphic.
- `images/feature_graphic_1024x500.svg`: editable source for the feature graphic.

Phone screenshots must be captured from a real app build so they accurately show the current app UI.
