RadarLite

A lightweight, free and open source Android speed camera warning app with no ads, no subscriptions, no user tracking, and no mobile data use while driving.

RadarLite passively listens for location updates already produced by other apps and alerts when an OpenStreetMap speed-camera record appears ahead. Once the camera database is installed, normal monitoring works offline; internet access is only needed to update the database.
