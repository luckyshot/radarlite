RadarLite Privacy Policy

Last updated: 16 June 2026

Summary

RadarLite does not include ads, subscriptions, analytics, crash reporting, user tracking, or third-party tracking SDKs.

RadarLite is free and open source. You can review the source code at github.com/luckyshot/radarlite.

Data Processed On Device

RadarLite processes passive location, speed, and bearing updates on your device to detect nearby speed-camera records. Recent alert history is stored locally in the app database.

Your location and alert history are not transmitted to the project maintainers.

Network Requests

RadarLite uses the network only when checking for or downloading the camera database from GitHub Releases. Normal monitoring works offline and does not use mobile data. Database update requests are made to GitHub-hosted release URLs.

Permissions

RadarLite requests foreground and background location so it can receive passive location updates while running in the background. It does not start its own high-accuracy GPS polling.

Notification permission is used for the required foreground service notification on supported Android versions.

OpenStreetMap Attribution

Camera data is derived from OpenStreetMap. OpenStreetMap data is available under the Open Data Commons Open Database License. See OpenStreetMap copyright and license information.

Contact

For questions or issues, use the GitHub repository: github.com/luckyshot/radarlite.
